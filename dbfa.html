DBFA's download package has a 'DBFApip' file with all pip commands to straightly pip from.

Setup:

Install Python (recommended 3.7.4)
(select *get pip* and *add to PATH*)

Use this command to pip from the .txt file inside DBFA's package:
pip install -r DBFApip.txt

Run the program. If you encounter any issues/ crashes, check your environment and edit the code with absolute paths, though DBFA is made to auto-handle this.

NOTE: DBFA was natively-coded to run on Windows. Use modules compatible with your operating system if you encounter module issues.
